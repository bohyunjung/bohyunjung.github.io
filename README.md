# bohyunjung.com

Personal blog at [bohyunjung.com](https://bohyunjung.com), built with [Docusaurus](https://docusaurus.io/), deployed on [Vercel](https://vercel.com).

Posts are written in Korean (`blog/`) with English translations in `blog-en/`. The site auto-detects browser language on first visit and redirects accordingly, with the preference saved to `localStorage`.

## Setup

```bash
npm install
```

## Local development

```bash
npm start
```

## Build

```bash
npm run build
```
